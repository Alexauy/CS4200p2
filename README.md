# N-Queens Problem Project #
## Project Description ##
*This project implements the ***steepest-ascent hill-climbing algorithm*** and the ***genetic algorithm*** to solve the N-Queen problem for n values of 8, 12, and 16. 
In it, the board configuration is filled with n queens, and a board is solved once none of the queens are able to attack one another.
Both algorithms gradually move queen pieces in search of this configuration*

## How to run: ##
- Unzip folder
- Open project directory in an IDE
- Navigate to ProblemSolver.java
- Run ProblemSolver.main()
  
*It is possible, albeit very unlikely, that the algorithms will fill the printed sample solutions with a board configuration that has >0 conflicts.
Running the program again will likely ensure that the new sample board configurations generated are valid solutions.
(This was done to avoid arbitrarily running the algorithms until a solution with 0 conflicts could be found)*
